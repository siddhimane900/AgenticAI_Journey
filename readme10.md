# 🤖 Day 10 - Autonomous Planning Agent

## 📌 Overview
This project demonstrates an Autonomous Planning Agent that generates step-by-step plans to achieve user-defined goals.

## 🚀 Features
- Goal-based planning
- Predefined plans for common goals
- Generic planning for custom goals
- Fully offline and free to use
- Beginner-friendly implementation

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
```bash
python planning_agent.py
