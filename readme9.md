# 🤖 Day 9 - Multi-Tool Agent

## 📌 Overview
This project demonstrates a Multi-Tool Agent built using Python. The agent can perform calculations, tell jokes, provide the current date and time, and remember user information using persistent memory.

## 🚀 Features
- 🧮 Calculator for arithmetic operations
- 🕒 Current date and time
- 😂 Random joke generator
- 🧠 Persistent memory using JSON
- 💬 Interactive command-line interface
- 💰 Fully offline and free

## 🛠️ Technologies Used
- Python 3
- JSON
- OS Module
- Random Module
- Datetime Module

## ▶️ How to Run
```bash
python multi_tool_agent.py
