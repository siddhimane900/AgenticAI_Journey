# 🤖 Day 8 - File-Based Memory Agent

## 📌 Overview
This project demonstrates a Memory-Based Agent that stores user information in a JSON file, enabling persistent memory across sessions.

## 🚀 Features
- Remembers the user's name
- Stores favorite programming language
- Persistent memory using JSON
- Personalized greetings
- Works completely offline

## 🛠️ Technologies Used
- Python 3
- JSON for data storage

## ▶️ How to Run
```bash
python memory_agent.py
