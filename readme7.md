# 🤖 Day 7 - Tool-Using Agent (Calculator)

## 📌 Overview
This project demonstrates a simple Agentic AI concept where a chatbot can use a calculator tool to perform mathematical operations based on user input.

## 🚀 Features
- Performs mathematical calculations using the "calculate" command
- User-friendly greeting and help instructions
- Handles invalid inputs gracefully
- Works completely offline
- Beginner-friendly implementation

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run

```bash
python tool_agent.py
