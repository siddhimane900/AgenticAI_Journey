# 🤖 Day 11 - LLM-Based Agent

## Overview
This project demonstrates a simple LLM-based conversational agent using the Mistral model via Ollama. The agent runs completely offline and does not require any API keys.

## Technologies Used
- Python
- Ollama
- Mistral LLM

## How to Run
1. Install Ollama from https://ollama.com
2. Pull the model:
   ollama pull mistral
3. Run the script:
   python llm_agent.py
